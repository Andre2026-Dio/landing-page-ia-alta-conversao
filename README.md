# Estratégia de Alta Conversão: IA + Psicologia Comportamental

Este projeto apresenta o desenvolvimento de uma Landing Page estratégica onde utilizei **Inteligência Artificial Generativa** para criar uma estrutura de vendas baseada em gatilhos mentais e psicologia do consumidor.

## 🚀 Resultados Alcançados
* **Custo por Clique (CPC):** R$ 0,10.
* **Otimização:** Redução de custos através de Engenharia de Prompt para anúncios segmentados.

## 🧠 Diferencial Estratégico
Como psicólogo, apliquei conceitos de **Psicologia Comportamental** para estruturar o copy (texto):
1. **Gatilhos Mentais:** Autoridade e Reciprocidade.
2. **Análise de Público:** Segmentação baseada em dores e necessidades reais.
3. **Engenharia de Prompt:** Uso de IAs para iterar versões de anúncios até atingir a performance máxima.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **IAs Generativas:** Google Gemini e ChatGPT (Criação de Copy e roteiros).
* **Design e Landing Page:** Canva e WordPress.
* **Engenharia de Prompt:** Estruturação de comandos para anúncios de alta performance.
* **Gestão de Tráfego:** Monitoramento de métricas e otimização de orçamento.
